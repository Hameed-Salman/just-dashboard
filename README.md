# JUST University Admin Dashboard

## Overview
This is a web-based admin dashboard used to manage JUST University system data.

It allows authorized administrators to update university information, upload images, and manage system content.  
The dashboard communicates directly with the REST API.

---

## Features

- Admin login system using JWT authentication
- Add and update university information (name, description, location)
- Upload images to Cloudinary via API
- Create new admin users
- Real-time data updates from backend API
- Simple and responsive UI using Tailwind CSS

---

## Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- REST API Integration

---
